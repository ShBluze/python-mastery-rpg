## 🎯 Objetivos

Al completar este módulo podrás:
- Leer y escribir archivos en múltiples formatos (CSV, JSON, SQLite)
- Navegar y manipular el sistema de archivos con pathlib
- Escribir tests con pytest que cubran tu código
- Configurar tooling moderno (ruff, mypy, pre-commit)
- Implementar logging estructurado y debugging efectivo
- Empaquetar y publicar tu propio paquete Python
