# 🧮 Mini-Proyecto 01 — Calculadora CLI

**Dificultad**: 🟡 Medio | **XP**: 100 | **Módulo**: 01-fundamentos

---

## 📖 Historia

Eres un aprendiz de mago Python y necesitas construir un artefacto mágico que realice cálculos aritméticos. Los aldeanos de la región te han pedido una calculadora que funcione desde la terminal.

## 🎯 Requisitos funcionales

- [ ] Menú interactivo con opciones: sumar, restar, multiplicar, dividir, potenciar, raíz cuadrada, historial, salir
- [ ] El usuario ingresa dos números (o uno para raíz)
- [ ] Muestra el resultado con formato claro
- [ ] Guarda historial de operaciones en una lista (en memoria)
- [ ] Opción para ver historial
- [ ] Manejo de errores: división por cero, entrada inválida, números negativos en raíz

## 📋 Criterios de éxito

- El programa no crashea bajo ninguna entrada
- El menú se repite hasta que el usuario elige salir
- El historial muestra: operación, operandos, resultado
- Al menos 3 funciones definidas (no todo en main)
- Código documentado con docstrings

## 🧪 Ejemplo de uso

```
=== CALCULADORA MÁGICA ===
1. Sumar
2. Restar
3. Multiplicar
4. Dividir
5. Potencia
6. Raíz cuadrada
7. Historial
8. Salir
Elige: 1
Primer número: 10
Segundo número: 5
Resultado: 10 + 5 = 15
```
