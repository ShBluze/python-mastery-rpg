## 🎯 Objetivos

Al completar este módulo podrás:
- Escribir scripts funcionales con variables, condicionales y bucles
- Usar listas, diccionarios, tuplas y sets correctamente
- Definir y llamar funciones con parámetros flexibles
- Manejar errores con try/except
- Organizar código en módulos reutilizables
- Documentar tu código profesionalmente

1. Lee cada archivo de teoría en orden (01 → 09)
2. Ejecuta los ejemplos de [`ejemplos/`](ejemplos/)
3. Resuelve los ejercicios de [`ejercicios.md`](ejercicios.md)
4. Comprueba tu HP en `../rpg-system.md`
5. Cuando tengas HP ≥ 50, enfrenta el boss en [`boss-fight/`](boss-fight/)
6. Si pasas: ¡siguiente módulo desbloqueado!
