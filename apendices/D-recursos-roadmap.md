## Herramientas del ecosistema Python

### Gestión de entornos

| Herramienta | Cuándo usarla |
|-------------|---------------|
| `venv` | Built-in, simple, para proyectos pequeños |
| `poetry` | Dependencias + packaging moderno |
| `uv` | Extremadamente rápido (Rust), 2024+ |
| `rye` | Gestión unificada (pip + venv + toolchains) |
| `conda` | Data Science, Windows, dependencias no-Python |
| `pipenv` | Alternativa a poetry (menos popular hoy) |

### Linters y formateadores

| Herramienta | Rol |
|-------------|-----|
| `ruff` | Linter + formateador ultra-rápido (Rust) |
| `mypy` | Type checker estático |
| `pyright` / `basedpyright` | Type checker alternativo (más rápido) |
| `pre-commit` | Hooks automáticos antes de commit |

### Testing

| Herramienta | Rol |
|-------------|-----|
| `pytest` | Framework de tests estándar |
| `pytest-cov` | Cobertura de código |
| `tox` / `nox` | Tests multi-versión |
| `hypothesis` | Property-based testing |
